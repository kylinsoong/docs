man

man 种类，manDB
$ man man
$ man 1 man
$ man 7 man

$ man -k partition
$ man -k adduser
adduser (8)          - create a new user or update default new user information
$ man 8 adduser

date

查看当前日期和时间
$ date
Sun Nov 19 20:55:25 CST 2017
$ date +%x
11/19/2017
$ date +%R
20:57

file

查看文件类型
$ file /etc
/etc: directory
$ file /etc/passwd
/etc/passwd: ASCII text

head

显示文件前 10 行
$ head /etc/passwd
root:x:0:0:root:/root:/bin/bash
bin:x:1:1:bin:/bin:/sbin/nologin
daemon:x:2:2:daemon:/sbin:/sbin/nologin
adm:x:3:4:adm:/var/adm:/sbin/nologin
lp:x:4:7:lp:/var/spool/lpd:/sbin/nologin
sync:x:5:0:sync:/sbin:/bin/sync
shutdown:x:6:0:shutdown:/sbin:/sbin/shutdown
halt:x:7:0:halt:/sbin:/sbin/halt
mail:x:8:12:mail:/var/spool/mail:/sbin/nologin
operator:x:11:0:operator:/root:/sbin/nologin
显示文件前 3 行
$ head -n 3 /etc/passwd
root:x:0:0:root:/root:/bin/bash
bin:x:1:1:bin:/bin:/sbin/nologin
daemon:x:2:2:daemon:/sbin:/sbin/nologin

wc

统计文件中行数，字数，字符数
$ wc /etc/passwd
  41   79 2107 /etc/passwd
$ wc -l /etc/passwd
41 /etc/passwd
$ wc -w /etc/passwd
79 /etc/passwd
$ wc -c /etc/passwd
2107 /etc/passwd

locate

locate 用来文件定位。

locate
# locate Kylin
locate -i
# locate -i Kylin
locate -i -n 1
# locate -i -n 1 Kylin
/home/kylin
updatedb
# updatedb
Note
 默认 locate 数据库会每天更新一次，updatedb 可以进行手动更新。

find

find 命令用来在 Linux 文件系统上进行文件查找。

find -name
# find / -name sshd_config
# find / -name '*.txt'
# find /etc -name '*pass*'
find -user 查找某用户拥有的文件
find / -user kylin
find -iname
# find / -iname '*messages*'
find -size
# find / -size 10M
# find / -size +10G
# find / -size -10k
find -mmin
# find / -mmin 120
# find / -mmin +120
# find / -mmin -120
find -perm
# find /home -perm 764
find -type
# find /etc/ -type d
# find /etc/ -type f
# find / -type l
# find /dev/ -type b
find -links
# find / -type f -links +1
